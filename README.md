# Campus Placement Prediction & Analysis

## Overview
This is my first data analytics and machine learning project where I analyzed student placement data and built predictive models.

## Objective
To predict student placement based on academic and skill-based features.

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## Models Used
- Logistic Regression
- Support Vector Machine (Linear & RBF)

## Results
- Logistic Regression Accuracy: ~80%
- SVM (Linear): ~80%
- SVM (RBF): ~80.9%

## Key Insights
- Placement depends on multiple factors, not just CGPA
- Aptitude, projects, and academic scores influence outcomes
- Dataset shows both linear and slight non-linear patterns

## Conclusion
This project demonstrates how machine learning can be applied to real-world data to generate insights and predictions.
